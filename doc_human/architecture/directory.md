# Directory Reference

Summarizes key folders and what they contain. Use it when orienting new contributors.

| Path | Description |
|------|-------------|
| `AGENTS.md` | Root orchestration config |
| `doc_agent/` | AI-friendly policies + quickstarts |
| `doc_human/` | Human guides, templates, examples |
| `modules/` | Feature modules (shared code in `modules/common`) |
| `db/engines/` | Schemas, migrations, DB docs |
| `observability/` | Logging/metrics/tracing configs |
| `scripts/` | Automation utilities |
| `tests/` | Cross-language test suites |

Remind newcomers about the language policy (`config/language.yaml`).

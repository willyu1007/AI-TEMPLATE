## 约束（性能/安全/合规）

### 性能约束
- API 响应时间：P95 < 200ms, P99 < 500ms
- 并发支持：1000+ 在线用户
- 数据库查询：单查询 < 50ms

### 安全约束
- 数据传输：全程 HTTPS
- 密码存储：bcrypt 加密
- 敏感数据：符合 GDPR/个人信息保护法
- 会话管理：JWT + Redis

### 合规约束
- 数据存储：[地区] 境内
- 日志保留：180 天
- 备份策略：每日备份，保留 30 天

### 技术约束
- 浏览器支持：Chrome 90+, Firefox 88+, Safari 14+
- 响应式设计：支持 1920x1080 到 375x667
- 离线功能：不支持（仅在线）
